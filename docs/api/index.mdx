# Claire-api

[Github repo](https://github.com/unlimited-hr/Claire-api)

The Claire API is the glue that holds everything together. It sends the measured air data to the database, and retrieves when the user wants to check on their home through the mobile app. It also communicates with the Claire artificial intelligence to generate relevant advice for the user.
